About

For my website, I created a marketing website for Coca-Cola. I added a 3D model of a can with an opening animation and sound effect, as well as a crushed version of the can.

3D Models

For this module, I created Coke, Sprite, and Dr Pepper cans, each on their own webpage. I built an animation for the Coke can where the ring pull and tab move as though opening the can. A crushed can model was also created as a separate animation.

For Sprite and Dr Pepper, I designed a different opening mechanism and unique animations showing the bottles being crushed.

Design

I integrated the can model within a responsive canvas using Bootstrap and added a navigation bar. Custom CSS was used to improve the interactivity and aesthetics of the interface. I used Three.js to apply a Hemisphere Light for ambient lighting, and a Spot Light with a dat.GUI panel for user-controlled lighting effects.

The 3D models were loaded using Three.js. I also created interactive buttons for users to open the can, switch models, and play the crushing animation.

Integration

I edited and enhanced audio using Adobe Audition, synchronizing it with the animations when they played on the website. I also added images to the landing page that resize responsively based on screen size using CSS.

Interaction

I implemented orbit controls so users can freely rotate the 3D model. A toggle button was added to switch between different models inside the Bootstrap canvas, making the experience more dynamic and interactive.

Implementation

I conducted several walkthroughs to test functionality and usability. The background color was modified to make the design feel more vibrant. I center-aligned the layout and bolded key text for better readability.

Deeper Understanding

Beyond the lab videos, I independently developed and integrated three completely different models. I refined the layout and style continuously, aiming for a unique and cohesive visual identity for the site.

Implementation and Publication

An About page was added to provide context. I zipped the website folder and tested it in Visual Studio Code to confirm that it works as expected after extraction.
